# CS366-FinalProject-FinacialLiteracy
Includes prompting for GPT customization and configuration of both Narrative and Factual conditions, SPSS output files, and link to interactive Figma prototype.
Instructions on how to run Figma prototype is listed below.

To run Figma prototype on your own desktop: 
1. Open the following link on your web browser: https://www.figma.com/proto/dhSWeU8HuwOVtDUUqhADWG/Financial-Literacy-Learning-Game?node-id=1-2&t=SXPrcNrRGntEsQXC-1
2. Click on either Factual or Narrative buttons to start the game
3. Continue to click on the screen one click at a time to progress through the game.
4. When there's an option to choose between A or B buttons, click on either button once to choose your answer.
5. Use the left and right arrow keys (shows up when screen is clicked on but not on any interactive button) to move back or forward through the game.
